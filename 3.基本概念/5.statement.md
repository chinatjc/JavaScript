### 语句

#### if 语句
```javascript
    if (condition1) {
        statement1
    } else if (condition2) {
        statement2
    } else {
        statement3
    }
```
> 针对condition1、condition2，如果不是Boolean类型的值，if语句会使用Boolean( )方法转换为Boolean类型的值

<br>

#### do-while语句
```javascript
    do {
        statement
    } while (condition);
```
- do-while语句是一种后测试循环语句
- 在判断条件之前，执行一遍语句
```javascript
    var i = 0;

    do {
        i++;
    } while (i < 10);

    // i -> 10
```

<br>

#### while 语句
```javascript
    while (condition) {
        statement
    }
```
- while语句是一种前测试循环语句
- 在满足判断条件之后，才决定是否执行语句
```javascript
    var i = 0;

    while (i < 10) {
        i++;
    }

    // i -> 10
```

<br>

#### for 语句
```javascript
    for (initialization; expression; post-loop-expression) {
        statement
    }
```
- for语句是一种前测试循环语句
- initialization, expression, post-loop-expression，三个参数都可以省略
- expression，默认为true，也就是说省略expression，可以实现无限循环
- for语句执行顺序
    - 1. initialization
    - 2. loop body -> expression&nbsp;&nbsp;&nbsp;?&nbsp;&nbsp;&nbsp;statement;&nbsp;post-loop-expression;&nbsp;&nbsp;&nbsp;:&nbsp;&nbsp;&nbsp;end-loop;
    - 3. end
```javascript
    // for语句执行顺序

    {
        // 初始化
        initialization;

        // 判断 & 循环
        if (expression) {
            // 变量差异化，形成闭包
            // 块级作用域内声明变量i，并且赋值为上一作用域
            // let i = i;
            statement;
            post-loop-expression;
        }

        if (expression) {
            // 变量差异化，形成闭包
            // let i = i;
            statement;
            post-loop-expression;
        }
    }
```

```javascript
    var count = 10;

    for (var i = 0; i < count; i++) {
        console.log(i);
    }

    // i -> 10
```

#### for-in 语句
```javascript
    for (property in expression) {
        statement
    }
```
- for-in语句是一种精准的迭代语句，可以用来枚举对象自有的、继承的、可枚举的、非Symbol的属性
- for-in语句，不能保证 `枚举对象属性顺序的一致性`
- for-in语句，在遍历对象的过程中，只读对象即可，不要增删对象的属性
- for-in语句，建议使用 var 操作符，保证是局部变量，而不是全局变量，污染全局环境
```javascript

    for (var property in window) {
        console.log(property);
    }

    // for-in语句可以枚举所有可以通过对象访问到的、可枚举的属性
    function Person() {
        this.name = 'Mike';
    }

    Person.prototype.sayName = function() { return this.name1 };

    Object.defineProperty(Person.prototype, 'name1', { value: 'John' });

    var person = new Person();

    Object.defineProperty(person, 'job', { value: 'fe' });

    for (var p in person) {
        console.log(p);
    }
    // name sayName    name1和job因为不是可枚举的属性，所以无法通过for-in语句枚举

```

#### label 语句
```javascript
    lableName: statement
```
- 用 labelName 给代码中的语句添加 标签
- label添加的语句标签，供break、continue语句使用：该停止哪个循环
```javascript

    loop1: for (var i = 0; i < 10; i++) {
        for (var j = 0; j < 10; j++) {
            if (i === 5 && j === 5) {
                break loop1;
            }
        }
    }

    // i -> 5
    // j -> 5


    for (var i = 0; i < 10; i++) {
        for (var j = 0; j < 10; j++) {
            if (i === 5 && j === 5) {
                break;
            }
        }
    }

    // i -> 10
    // j -> 10

```

<br>

#### break 语句
- 使用break语句，会立即退出 `break所处的循环队列中最内层` 循环，执行循环后面的语句
- 配合label语句，可以控制break退出哪个循环
```javascript

    for (var i = 0; i < 10; i++) {
        for (var j = 0; j < 10; j++) {
            if (i === 5 && j === 5) {
                break;
            }
        }
    }

    // break只是退出了 for(var j 的循环，没能阻止 for(var i 的循环
    // i -> 10
    // j -> 10

    loop1: for (var i = 0; i < 10; i++) {
        for (var j = 0; j < 10; j++) {
            if (i === 5 && j === 5) {
                break loop1;
            }
        }
    }

    // break loop1，彻底阻止了循环
    // i -> 5
    // j -> 5

```

<br>

#### continue 语句
- 使用continue语句，会立即退出 `continue所处的循环队列中最内层` 循环的这一次循环，但还会执行这层循环(如果可以循环的)
- 配合label语句，可以控制continue退出哪个循环
```javascript

    var num1 = 0;

    for (var i = 0; i < 10; i++) {
        for (var j = 0; j < 10; j++) {
            if (i === 5 && j === 5) {
                continue;
            }
            num1++;
        }
    }

    // continue只是退出了 for(var j 循环中，i === 5 && j === 5的这一次循环，没能阻止 for(var j 循环的继续执行
    // num1 -> 99

    var num2 = 0;

    loop1: for (var i = 0; i < 10; i++) {
        for (var j = 0; j < 10; j++) {
            if (i === 5 && j === 5) {
                continue loop1;
            }
            num2++;
        }
    }

    // continue退出了 for(var i 循环中，i === 5 && j === 5的这一次循环(损失 i === 5 时，j -> 5~9，这5次的计数)，没能阻止 for(var i 循环的继续执行
    // num2 -> 95

```

#### switch 语句
```javascript
    switch (expression) {
        case value:
            statement;
            break;
        case value:
            statement;
            break;
        case value:
            statement;
            break;
        default:
            statement;
    }
```
- 当 expression === value 时，执行case对应的statement，遇到break跳出switch，否则继续执行下一个case
```javascript

    switch ((new Date()).getDay()) {
        case 1:
            console.log('星期一');
            break;
        case 2:
            console.log('星期二');
            break;
        case 3:
            console.log('星期三');
            break;
        case 4:
            console.log('星期四');
            break;
        case 5:
            console.log('星期五');
            break;
        case 6:
            console.log('星期六');
            break;
        case 7:
            console.log('星期日');
            break;
    }

    // 合并case
    switch ((new Date()).getDay()) {
        case 1:
            console.log('周一');
            break;
        case 2:
            console.log('周二');
            break;
        case 3:
            console.log('周三');
            break;
        case 4:
            console.log('周四');
            break;
        case 5:
            console.log('周五');
            break;
        case 6:
        case 7:
            console.log('周末');
            break;
    }

```
- 当 expression === value 时，执行case对应的statement， `若statement语句中没有break` ，则 `该case下面的case无需判断，依次执行case中的statement`
```javascript
    var text = '';

    switch ((new Date()).getDay()) {
        case 1:
            text = '周一';
        case 2:
            text = '周二';
        case 3:
            text = '周三';
        case 4:
            text = '周四';
        case 5:
            text = '周五';
        case 6:
        case 7:
            text = '周末';
    }

    console.log(text); // '周末'    不论是哪一天，最终的text为'周末'
```
